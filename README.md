# Flow Builder

A Python-based flow builder application.

## Prerequisites

- Python 3.14
- [uv](https://docs.astral.sh/uv/) package manager

## Installation

### 1. Install uv

First, install `uv` package manager by following the instructions at [https://docs.astral.sh/uv/](https://docs.astral.sh/uv/)

### 2. Clone the Repository

```bash
git clone https://github.com/Taufik041/flow_builder.git
cd flow_builder
```

### 3. Install Dependencies

```bash
uv sync
```

### 4. Run the Application

```bash
uv run main.py
```
